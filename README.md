BCS601 Cloud Project – Simple Calculator

Project Title

Simple Calculator Web Application using Flask (Cloud Deployed)

Introduction

This project is a simple web-based calculator developed using Python Flask framework.
It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division through a user-friendly interface.

The application is deployed on a cloud platform, demonstrating practical knowledge of cloud computing and web deployment.

Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division

Technologies Used

Frontend: HTML, CSS
Backend: Python (Flask)
Deployment: Render / Google Cloud
Version Control: GitHub
